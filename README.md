# Сервис страниц памяти

Однопользовательский CMS для создания мемориальных страниц с QR-кодами.

## Быстрый старт

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py load_demo   # опционально: демо-страница
python manage.py runserver
```

- **Панель управления:** http://127.0.0.1:8000/panel/
- **Демо-страница:** http://127.0.0.1:8000/p/thompson-margaret-elaine-1948/

## Возможности

- Создание/редактирование/удаление страниц памяти
- Генерация slug из ФИО + год рождения (транслитерация)
- Секции: портрет, биография, события, галерея, таймлайн, воспоминания
- Слайд-шоу с музыкой из блоков биографии
- Форма воспоминаний от посетителей с модерацией
- Генерация QR-кода для печати на табличке
- Кнопки «Поделиться» (VK, Telegram, копировать ссылку)
